# Student Bank Details Collection Portal

A modern, highly responsive, and interactive web application designed to collect and manage student bank details securely.

## Features

- **Dynamic Form**: Captures student name, mobile number, bank name, account number, and IFSC code.
- **Real-Time Input Validation**: Checks fields dynamically (e.g., IFSC formatting, digit length, mobile matching).
- **Confirmation & Receipts**: Shows a summary of the input upon submission and generates a downloadable text receipt.
- **Admin Dashboard**: Access the mock database with a PIN (`1234` by default) to view all submitted records.
- **Search & Filter**: Find submissions instantly by name, mobile, or bank.
- **Export Capabilities**: Download records as standard CSV or JSON format files.
- **Dark & Light Mode**: Fluid transition between dark and light themes.

## How to Run

1. Open [index.html](file:///c:/StudentBankDetails/index.html) in any modern web browser.
2. Fill out the student bank details to test validation and submission.
3. Access the administrative view by clicking the "Admin Dashboard" button in the upper-right corner and entering the access code `1234`.
